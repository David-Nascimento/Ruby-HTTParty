# Example
Nflix é uma gem de testes criada pela QANinja para testes de automação Front e API

## Installation
Versão minima do Ruby => 2.7.6

Adicionar a linha no seu Gemfile
```
gem 'nflix'
```

E então execute:

    $ bundle install

Ou intale você mesmo:

    $ gem install nflix

## Forma de utilizar
```ruby
    require 'nflix'

    nflix.delete_user(email)
```
